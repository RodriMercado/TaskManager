# Task Manager

A web application developed with **Django** for managing personal tasks.
It allows users to create, edit, complete, and delete tasks in a simple and intuitive way.

---

## 🚀 Features

* User registration and authentication
* Create tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Each user can only view and manage their own tasks

---

## 🛠 Technologies Used

* **Python**
* **Django**
* **SQLite** (default database)
* **HTML + Bootstrap**
* **Git & GitHub**
